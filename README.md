# Branch Turnaround Case Study (SQL)

This repository contains a real-world branch turnaround analysis built using SQL Server.

The objective is to demonstrate:
- Analytical SQL (YoY analysis, rank movement)
- Clean data modeling and ER design
- Attribution of business actions to ranking outcomes

This repository supports a detailed Medium case study (to be published).
